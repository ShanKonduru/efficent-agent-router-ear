# EAR v0.10.3 Release Notes (Draft)

> Note: Tag `0.10.1` was created 2026-05-01 against an earlier commit. This draft tracks the current release head at `0.10.3`.

## Summary

v0.10.3 consolidates planning and demo updates after v0.10.2, including a live OpenRouter LLM Explorer UI and roadmap expansion for E10 and E11.

## Highlights

- Added `docs/llm_explorer.html`: live OpenRouter model explorer for leadership and investor demos.
- Added auto-refresh and cache-busting model fetch behavior with last-updated indicator.
- Added sortable table columns and Excel-style per-column filtering.
- Added free vs paid radio filtering and selectable side-by-side model comparison cards.
- Expanded planning docs with E10 (execution runtime + adaptive routing intelligence) and E11 (demo frontend) structure.
- Updated project version to `0.10.3` and synchronized `master`/`main` with release tag `0.10.3`.
